# react-spectrum-sample

A sample project demonstrating [Adobe React Spectrum](https://react-spectrum.adobe.com/react-spectrum/), a collection of libraries and tools that help you build adaptive, accessible, and robust user experiences.

## Overview

This project showcases basic React Spectrum components including:

- **Provider** - Theme provider for React Spectrum applications
- **Button** - Standard button with CTA variant
- **TextField** - Text input with label support
- **ActionButton** - Action button for form submissions

## Prerequisites

- Node.js (v14 or higher recommended)
- npm or yarn

## Installation

```bash
git clone https://github.com/conao3/react-spectrum-sample.git
cd react-spectrum-sample
npm install
```

## Usage

Start the development server:

```bash
npm start
```

Build for production:

```bash
npm run build
```
